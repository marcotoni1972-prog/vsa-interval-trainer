# VSA — Tavola degli Intervalli Naturali

Allenatore interattivo per lo studio degli intervalli musicali secondo il metodo VSA.

## Come funziona

L'app legge la **Tavola degli Intervalli Naturali** riga per riga (12 righe × 30 numeri = 360 domande totali).

Per ogni numero visualizzato devi rispondere con la nota corrispondente a quel grado nella tonalità scelta.

- La tonalità viene scelta casualmente tra le 12 (sempre con il bemolle, mai il diesis)
- Inserisci la nota nella casella di testo e premi Invio o OK
- L'app ti dice subito se hai risposto correttamente
- Puoi mostrare/nascondere la scala di riferimento in qualsiasi momento
- Il tuo punteggio viene aggiornato in tempo reale

## Uso

Apri semplicemente `index.html` in un browser. Nessuna dipendenza, nessun server necessario.
